# Claude Design bundle (snapshot)

This folder is a **frozen copy** of selected Quni source files used to **extract and document the existing design language** (tokens, Tailwind usage, admin patterns, verification badges, listing cards) inside **Claude Design**. It mirrors repo paths under `src/` so filenames stay recognizable—for example `src/pages/admin/` vs `src/components/` shows composition, not a flat dump.

**Snapshot date:** 2026-05-10

**Important:** These files are **not** the live source of truth. Edit and ship changes in the real paths (`tailwind.config.js`, `src/…`) in the project root. Regenerate this bundle when you want an updated paste or upload for design tooling.
